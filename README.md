# Filmly Mobile

Filmly Mobile is a mobile application for movie recommendations and movie rating.

## Development

Filmly mobile is developed using React Native with Expo framework.

Application is using backend api from Filmly Web application which is available on [Github](https://github.com/domi-04/filmly-web).

### Usage

Before starting, it is necessary to setup the environment for develoing React Native applications.
Setup is available at the [link](https://reactnative.dev/docs/set-up-your-environment).

After cloning this repository, you must install npm dependencies by running following command inside project directory:

```bash
npm install
```

Then, you only need to run expo project by using following command:

```bash
npx expo start
```
